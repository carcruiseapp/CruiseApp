# Cruise Public Support Site

This folder contains ready-to-publish pages for:

- `privacy-policy.html`
- `support.html`

## Before Publishing

The support email in these files is already set to:

- `CarCruiseApp@gmail.com`

## Free Hosting Options

You can host these pages for free on:

1. GitHub Pages
2. Netlify
3. Cloudflare Pages

## Recommended Final URLs

If you host this folder, use:

- Privacy Policy URL: `https://YOUR-SITE/privacy-policy.html`
- Support URL: `https://YOUR-SITE/support.html`

## App Store / App Use

Once hosted, use those live URLs in:

- App Store Connect Privacy Policy URL
- App Store Connect Support URL
- the in-app Settings screens if you later want them to open the hosted pages
